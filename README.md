# MoreDisc (Fabric 1.21.11)

Mod Fabric qui ajoute 8 disques musicaux craftables.

## Build

```bash
./gradlew build
```

## Lancer le client de dev

```bash
./gradlew runClient
```

## Dossiers importants

- Java mod main: `src/main/java/fr/chavi/moredisc/`
- Entrypoint client: `src/client/java/fr/chavi/moredisc/`
- Descriptions des morceaux (jukebox): `src/main/resources/data/moredisc/jukebox_song/`
- Recettes de craft: `src/main/resources/data/moredisc/recipe/`
- Déclaration des sons: `src/main/resources/assets/moredisc/sounds.json`
- Fichiers audio OGG: `src/main/resources/assets/moredisc/sounds/records/`
- Textures des disques: `src/main/resources/assets/moredisc/textures/item/`
- Modčles d'items: `src/main/resources/assets/moredisc/items/` et `src/main/resources/assets/moredisc/models/item/`
- Langues: `src/main/resources/assets/moredisc/lang/`
