# 🎵 MoreDisc

> Ajoute plein de nouveaux disques musicaux à Minecraft !

**MoreDisc** est un mod qui enrichit l’expérience musicale de Minecraft en ajoutant de nouveaux disques à collectionner et à écouter dans ton jukebox.

⚠️ Le mod est actuellement en **BÊTA**.

---

## ✨ Contenu actuel

Pour le moment, le mod inclut :

* 🎄 Une musique de Noël de **Mariah Carey**
* 🎤 Le célèbre “For Sure” de **Emmanuel Macron**
* 🏠 Des musiques House
* 🎶 D’autres musiques variées (mèmes, ambiance, etc.)

De nouveaux disques seront ajoutés progressivement !

---

## 🚧 État du projet

MoreDisc est en **phase BÊTA** :

* Certaines fonctionnalités peuvent être modifiées
* De nouvelles musiques seront ajoutées
* Des bugs peuvent être présents

N’hésitez pas à signaler les problèmes sur GitHub ou Discord.

---

## 📢 Proposer une musique

Vous avez une idée de musique à ajouter ?

Rejoignez notre serveur Discord et envoyez :

* 🎵 Le nom de la musique
* 👤 Le nom de l’auteur (si possible – non obligatoire pour les mèmes)
* 📁 Le fichier audio **OU** au minimum un lien YouTube

Les suggestions sont les bienvenues !

---

## 🛠 Installation

1. Installez Minecraft compatible avec la version du mod
2. Installez le loader `Fabric` avec `Fabric API`
3. Placez le fichier `.jar` dans le dossier `mods`
4. Lancez le jeu et profitez 🎧

---

## 📌 Objectif du mod

L’objectif de MoreDisc est simple :
Donner plus de variété musicale dans Minecraft et permettre aux joueurs de personnaliser l’ambiance de leur monde.
